    Introdução e fundamentos ( O que são GIT e GITHUB? )

    Git é um dos conhecimentos básicos que você vai encontrar em qualquer trilha de estudos de programação. Não importa se você está estudando para ser frontend, backend, mobile, enfim… git é um conhecimento fundamental para trabalharmos com versionamento de código. Pensando naqueles que estão iniciando, hoje vamos entender o que é o Git, para que usamos e os conceitos básicos que você precisa saber para começar a versionar o seu código. Bora pro video?

Mas acho que o primeiro ponto que temos que focar, é entender qual o sentido de versionar o seu código e acho que a partir disso fica mais fácil perder o medo.
